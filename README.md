# Ginit

## How to use

## Testing

Use these command to run the tests

```bash
task test
```

```bash
make test
```

## Coverage

- create a `coverage` folder
- Use these command to see the coverage

```bash
task coverage
```

```bash
make coverage
```

- Open coverage/coverage.html to trace the coverage

## Benchmarks

Use these command to run the tests

```bash
make benchmarks
```

```bash
task benchmarks
```
